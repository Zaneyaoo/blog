# 一、两场筵席死与生
这一段故事，记载耶稣的神迹，表明死亡和生命之间的关系，也让我们看到信心跟疑惑之间的挣扎，这甚至在门徒里面是一个循环。.

第一到十三节约翰被杀，讲的是在福音中操练死亡，而这个部分讲的是在福音当中认识生命。这一段的重心，是在旷野喂饱五千人的神迹，这个故事是关乎生命的。约翰之死，是马太福音第一个部分的结束，也是第二个部分的开始。那么，他也给我们跟随耶稣的人，包括去读这个福音书的人一个心理上的预备，就是在约翰被杀后，我们开始预备好要去迎接福音书第二个部分的高峰，那就是耶稣的受苦和他的死。所以，在旷野喂饱五千人和海面行走这两个神迹里，你会发现有一个时间和一个转折点，就是耶稣离开众人，独自退到旷野，独自退到山上去，他要一个人祷告。它发生在约翰被杀这一个转折点，这对前面是一个总结，对后面是一个预告。

我们也可以看到另外一个对照，同样都是君王的筵席：希律王的筵席上，君王忽然就变脸在筵席上杀人；而在旷野中另一位君王的筵席，喂饱了五千人（还不算妇女儿童）。两场君王的筵席，一场发生在宫廷中，一场发生在旷野；一场发生在什么都有的地方，一场发生在什么都没有的地方；筵席上的君王，一位是地上的，另一位是从天上来到地上。

我们再看一幅画面：就是以色列人出埃及后在旷野中，神在什么都没有的地方，来喂养他的百姓，旷野当中的粮，成为以色列人对未来弥赛亚的一个盼望。旷野当中哪来的粮呢？不是从地里产出来的，一定是从天上掉下来的。以色列人曾经经历过，神借着摩西，用吗哪来喂饱在旷野当中的以色列人，他自己的百姓。所以，在两约之间，以色列人一直有一个对弥赛亚的盼望，那就是在旷野当中将降下粮食来。在两约之间的一些文献里面，不断地描绘说，到了那个时候，那一天要来，就是弥赛亚的日子要来。那一天来，有一个非常重要的记号，就是我们曾经所经历的，耶和华上帝对我们的拯救。我们知道，以色列人可能从孩子三岁的时候开始，妈妈就给他讲旧约的故事，他从小就知道这件事，可是，他会对妈妈说，现在我们很荒凉，我们会被欺负，现在神在哪里呢？他可能从小都是这样在盼望，所以说那个时候的记号，就是天上将再一次地降下吗哪，使那些饥饿的以色列人得以饱足。所以，在整个旧约当中，那饥饿的人将要得以饱足，就成为对救恩的一个指望。诗篇一百四十五篇第十五、十六节：“万民都举目仰望你；你随时给他们食物。你张手，使有生气的都随愿饱足 ”。阿们！

# 二、旷野吃喝赖恩典
你会看到一点，门徒们来见耶稣时说，这里是野地，没吃的，而时候又已经过了，大家身上都没带东西，所以，天时地利人和全都没有。怎么办？实际上，门徒虽然来问耶稣，其实门徒有办法。门徒给耶稣提了一个建议：让他们走，自己买，自己吃，有钱的吃干的，没钱的吃稀的，反正让他们自己去解决。这是门徒向耶稣所提的一个建议。然而耶稣说什么呢？耶稣说了一句话，很有意思，有人说这一段的故事，不单单是指向耶稣基督的福音和他所赐的生命，而且也从这一点上的描绘看出，这个故事同样指向对门徒的呼召和使命。因为耶稣不是说：“没关系，我给他们吃”。耶稣对他们说：“你们给他们吃”。耶稣对门徒说，这是你们的事，你们给他们吃。门徒说，我们有什么呢？我们只有这么一点。最后收集了一下，有五个饼两个鱼。然后耶稣说了一句话：“拿过来给我”，所以，有人说：你有什么？我什么都没有。拿来给主，拿来给主！主就能够用它生出生命来，喂饱五千人。

耶稣怎么来喂饱这五千人呢？他有一个最简单的动作，却是最重要的动作——耶稣拿着五个饼，望着天祝福。为什么望着天祝福呢？因为食物是从天上来的，生命是从天上来的。如果你今天活在城里面，出了街就有饭馆，你中午会不会出去望天？不会。因为你觉得横向的关系，就足以解决问题了。所以人需要被带到旷野当中，到了旷野当中，你发现你生命当中真正的问题就是——总是在横向的层面想去得到，而只有纵向层面才能满足的事情。但是，现在当这个横向的层面全都给你断绝了，天时、地利、人和全都没有的时候，只有当我们被带入旷野，我们才有机会停止自我的欺骗，明白必须仰赖恩典来养活自己。

福音意味着什么呢？福音意味着我们有责任工作，来养活我们自己的身体，甚至供养我们家人的需要，但是，福音在这个地方意味着，你绝对没有能力喂饱你自己的灵魂。福音是让你意识到一件事情，你绝对不能够自己照看自己的灵魂，你没有办法喂饱你自己。万民都要举目仰望他，他随时给我们食物；你张手，你举目，你望天，他使有生气的都随愿饱足。所以，耶稣起初进到旷野四十天，也面对身体的饥饿，他回应魔鬼撒旦的试探说，“人活着不是单靠食物，而是靠神口中所出的一切话。”当他的先锋，那福音的先锋约翰被杀的时候，他再一次退回到旷野；当群众来跟随他，当他喂饱五千人后，他再一次离开他们，独自退到山上，退到旷野当中，去向父祷告，向天来仰望。耶稣经历了这样的生命历程，就是他将自己的生命置于一种没有依靠的状态，他将自己带到一个没有横向支持，甚至是众叛亲离的状态，以至于他单单靠神的话语而活，并且把这样的一个福音带给你，就是你也必须靠垂直关系上的上帝的恩典和供应而活。作为一个跟随主的门徒，你不能自己喂饱自己。你知道我们对恩典的抗拒常常的反应是什么？那就是你常常有一种自力更生的想法。我们讲了，在肉身上面自力更生、殷勤工作是对的，但是，我们常常会认为在道德上面、在灵魂上面、在心里的需求上面、在婚姻关系上面，都可以自力更生，会把自己照顾得很好，可以喂饱自己。而福音是要打破这个假象，甚至是打破这个自我的谎言。

我们常常是这样告诉自己的：不是我有问题，是环境有问题；不是我要改变，是我周围的那些人需要改变。但是，当你被带到旷野中去的时候，喂饱五千人——这是整个福音书里最有名的一个神迹。为什么最有名呢？因为这是唯一一个四福音书全都记载了的神迹，这是关乎生命的一个神迹。

我们再把这个神迹和启示录做一个对照。这个旷野里的筵席，总是指向最后的那个晚餐。最后的那个筵席，就是耶稣要将自己献上的那个筵席。在四福音书里面都记载了这个故事，但是只有在《约翰福音》当中，耶稣在喂饱五千人之后，他有一番讲论，他说：“我就是从天上降下来的粮，吃我的肉，喝我的血的，就有永生。”
让我们问这样两个问题：第一，是谁喂饱了五千人？你会说是耶稣，他是神的儿子。第二，在旷野当中，耶稣是用什么喂饱了五千人？是五个饼，两条鱼吗？显然不是。那个物质上的五个饼，两条鱼，也不是一个在魔术意义上的理解——他让那个饼变得越来越多！耶稣用什么来喂饱了这五千人呢？耶稣说，他用他自己喂饱了你们。他说：“我就是天上降下来的粮”（约6:41）。明明不是在吃这五个饼，两条鱼吗？但是主耶稣说；你在吃我的肉，喝我的血，而你们有福了，吃我的肉，喝我的血的人，就有永生。所以让我这样讲，第一是耶稣喂饱了你们，他是喂饱你们的那一位；第二用来喂饱你们的，是主耶稣自己。换言之，他既是大祭司，他是那位真正要献祭的人，同时，他又是那祭物，他自己就是那被献上去的物。所以，旷野当中的这场筵席，乃是指向最后的晚餐。

# 三、基督死亡赐生命
然而，这个旷野当中关乎生命被赐下的神迹，暗含了死亡的信息。当耶稣说你们吃的就是他的时候，预示着什么？就是约翰被杀所预示的那个结果——除非他被杀，否则他不能成为你们的粮食；除非他被杀，否则他不会成为从天上降下来，供应在地上的你们的食物和一次献上的祭。主是那供应者，主也是那食物，这就是福音的奥秘，这就是弥赛亚的奥秘——在生命和死亡之间的一个反转，耶稣以他的死亡来赐下生命。约翰之死，已经预言了他的死。他退到旷野当中，他对这一群人生出怜悯的心，他喂养他们，但是，他也借着喂养他们的肉身，显出他们的灵魂要被他喂养，而他们的灵魂被他喂养、被他救活的道路，是透过他的死亡。他透过他自己的饥饿，来喂饱他的百姓；他透过他自己的死亡，来赐下他的生命。

亲爱的弟兄姐妹，耶稣若是不渴，我们就没有生命的水喝；耶稣若是不饿，我们就不能从任何人那里得饱足，这就是在一个巨大的露天的宴会厅当中，所发生的一个生命的奇迹。一个简单的姿势，教导了众人生命的来源——你应当从哪里盼望你生命的来源——就是望着天。如果不望天，如果不祝福，这露天的盛宴就不会开始。如果没有耶稣的死，耶稣就不会成为你们的食。我们的生命是被恩典来喂养的，而恩典是从天上来的，耶稣是将天与地连接起来的桥梁，耶稣是将生命跟死亡连接起来的中保，耶稣是将神与人之间连接起来的道路。所以“拿过来给我”这句话和“望天祝福”这个动作，都指向将来启示录里面最后的那一个羔羊的婚宴。

# 四、操练死亡盼永生
最后，我要给大家一些应用。当我们明白，约翰之死与耶稣在旷野喂饱五千人之间的关系；当我们明白，在旧约当中神以吗哪来喂养他的百姓与跟耶稣在旷野当中来喂饱这五千人之间的关系；当我们明白，耶稣说他是天上降下来的粮，他需要透过死亡，把他自己给你们吃和给你们喝之间的关系，我们就看到上一次的讲道跟这一次的讲道、上一次的约翰之死跟这一次耶稣喂饱五千人一样——上一次证道我们说你要在福音当中怎么样呢——操练死亡。

操练死亡，是要经历基督的复活。除非你经历基督的复活，不然你不能操练死亡。对死亡的操练，是以复活为前提的。那么，今天我们的题目是，在福音当中来认识生命，同样，你怎样才能够认识生命呢？除非你经历主的十架，除非你在这一个丰盛的筵席的神迹当中看见了耶稣的死，除非你看见了耶稣的复活，否则你的生命中不能操练死亡。除非你看见了耶稣的死，否则你的生命当中没有生命的迹象。这就是在福音当中的一个翻转。就像我开始讲的这一个例子，除非你活在那一个生命当中，不然你不会有受苦的经历；除非你有受苦的经历，否则你不会经历到新的生命在你里面的成长！

所以，让我有一个具体的对应，操练死亡是在于盼望永生，永生是我们不断地默想死亡，这个默想死亡不是悲壮的，不是那个壮烈的，好像英雄主义的，是因为我们盼望永生。那么同样，认识生命，是在乎恨恶自己。当你开始认出这是神的儿子，当你瞩目于你脚下的风浪的时候，你就走不过去。好像彼得一样，当你单单地来瞩目前面的那一位耶稣的时候，你就能走过你人生当中的风浪，你轻看环境，你恨恶自己，你就有在耶稣基督里的生命。所以，福音意味着你必须要在舍己、顺服、奉献和亏损当中来操练你的生命。你看到耶稣用他自己喂饱了我们，你就恨恶你自己，你就恨恶你自己从前是一个向世界要求满足的人，你就恨恶自己是一个依靠自己来喂饱自己的人。我以前所有的工作的目的，我以前所有的赚钱的目的，我以前甚至所有的服侍的目的，都是为了喂饱我自己。但是，福音让我看见一件事情，我不能用任何的事情来喂饱我自己，我不能够用我的成就来喂饱我自己，我不能够用我的道德来喂饱我自己，我只能够仰望耶稣，他的死、他的十字架来喂饱我自己。所以，好像约伯说，现在我恨恶我自己，现在我厌恶我自己！只有开始厌恶自己的人，才会来认识生命，一个没有在福音当中学习厌恶自己的人，他一定会厌恶别人；一个在福音当中没有学习来跟自己过不去的人，他通常都跟别人过不去。

所以，最后我要给你们很具体的操练生命、恨恶自己的应用。很多时候很多弟兄姐妹说，王怡牧师，你讲道老是最后没有到底该怎么做的具体方法。我今天给你们几个非常具体的建议，就是如何认识生命在于否定自己，如何恨恶自己跟自己过不去的几个具体的建议。

第一个建议，经常违背自己的意愿，顺服一个具体的人。在你的生命当中，顺服神，一定体现为顺服你生命当中的某一个人，你常常预备着违背你自己的意愿去顺服他。你说这不是跟自己过不去吗？对！跟自己过不去，就有新的生命。

第二，我想鼓励大家，特别是我们当中的年轻人，当你选择工作的时候，永远不要以收入为第一位的考量。我不是说你完全不考虑收入，有两份工作，那份工作收入高一点，那份低一点，永远不要用收入做你第一位的考量。你说那不是跟自己过不去吗？是的，跟自己过不去！这不是跟钱有仇吗？对！就是跟钱有仇！就是要跟一个爱钱的自己有仇！就是要跟一个以今生作为自己生命目标的那个自我有仇！不然，你就没有耶稣基督里的新生命的成长。

# 五、认识耶稣恨自己

一位牧师讲的话，我觉得非常的好：怎么恨恶自己，怎么跟自己过不去呢，一笔钱不知道该要不该要？那就不要！一份辛苦的工作不知道你该做，还是不该做？那就做！有一份责任不知道是自己的，还是别人的？那就是自己的！有一件的事情不知道是自己错，还是别人错？那就是自己错！这就是跟自己过不去，这就是操练认识生命，这就是操练恨恶自己，这就是在神的面前说：“主啊！只有你能够喂养我，我不向这个世界要求饱足，我不向他人要求饱足，在婚姻当中我不向我的配偶要求饱足，在友谊当中我不向对方要求饱足，在公司里面我不向我的老板要求饱足，我只向你要求饱足！”
所以，我的整个生命就变成了天路的历程。你有什么？我什么都没有。拿来给主！这个神迹是由小变多，但是，你要明白这件事情，重心并不是由少变多。在旧约当中，先知以利亚逃到旷野，他同样经历这一幕。谁喂饱他？乌鸦，上帝用乌鸦来喂饱他的仆人。后来，以利沙用二十个饼，喂饱了一百人。由少变多，变少为多，包括以弱胜强，乃是指向那从无到有和死里复活的神。阿们！

最后，给大家一个劝勉——认识耶稣，恨恶自己。不要心疼自己，让主来心疼你；不要心疼你自己，舍己、让自己受亏损，主心疼这样的人；自己心疼自己的人，就不再得到主的心疼。不要喂饱自己，不要让这个世界来喂饱你，不要祈求从别人那里得到喂饱自己的东西，让主来喂饱你。如果世界已经把你喂饱了，就没有主来喂饱你了！阿们！

我们一起低头祷告：

主啊，我们感谢赞美主！你的十字架是我们认识生命的唯一的方法，你的十字架也是那新的生命在这一个败坏的世界当中呈现出来的唯一的道路，我们认识福音，我们来做你的门徒，求你呼召我们像呼召起初的彼得、约翰、雅各一样，舍了船、舍了网、别了父亲，主让我们来信靠你，让我们目光指向你不指向旁边的风浪，让我们的目光望向天，不望向我们现在旷野中的缺乏，主啊！我们向你张口，你就充满我们，我们向你伸手，你就使我们随愿饱足。主啊，求你使我们因认识你，我们想办法对付自己，而不是对付你，也不是对付我们身边的人；我们想办法，甘愿让自己受亏损，因为我们已经得到了在你里面的饱足。主啊，让那一个新的自我从里面出来，这是在十字架上的，这是真正的自我的实现，除非我否定那一个老我，否则我就无法实现你所赐给我的那个新我，除非我经历你把你献上的那一个十字架道路，否则我就不能来吃你的肉，喝你的血，而得永生。主啊，让我们像你的门徒们一样说，这真是神的儿子！这真是我生命当中的主！主啊，今生你带我们去赴十字架的筵席，那最后的晚餐，将来有一天你要带我们去赴羔羊的婚宴，感谢赞美神！听我们如此的祷告祈求，奉靠耶稣基督宝贵的圣名。

阿们。
